## Introduction

Welcome to the SeaCliff POS Chef Guide. As a chef, you use the Kitchen Display System (KDS) to view incoming orders, manage preparation, and update order status in real-time.

### Your Role
As a **Chef**, you can:
- View kitchen orders in real-time
- See order details and special instructions
- Update individual item status
- Mark items as preparing, ready, or completed
- View order priorities and timing
- Communicate order status to front-of-house

**Note**: You cannot create orders, process payments, manage menu items, or access administrative functions. Your focus is on preparing orders efficiently.

---

## Getting Started

### Logging In
1. Navigate to the SeaCliff POS login page
2. Enter your chef credentials (email and password)
3. Click "Login"
4. You'll be automatically redirected to the Kitchen Display System

### First Time Setup
After your first login:
1. Verify display is showing correctly
2. Test audio notifications (if enabled)
3. Familiarize yourself with the layout
4. Review any pending orders
5. Understand the color coding system
6. Ask questions if unclear

### Kitchen Display Overview
The KDS shows:
- **Pending Orders**: New orders awaiting preparation
- **In Progress**: Orders currently being prepared
- **Order Details**: Items, quantities, special instructions
- **Timestamps**: When orders were placed
- **Table Numbers**: For delivery coordination
- **Priority Indicators**: Urgent or time-sensitive orders

## Kitchen Display System (KDS)

### Display Layout

#### Order Cards
Each order appears as a card showing:
- **Order Number**: Unique identifier (e.g., ORD-001)
- **Table**: Table number where order is going
- **Time**: How long ago order was placed
- **Items**: All kitchen items for this order
- **Special Instructions**: Guest requests, allergies, modifications
- **Status**: Current preparation state

#### Color Coding

**Order Card Colors**:
- **White/Light**: Just received (0-5 minutes)
- **Yellow**: Standard time (5-15 minutes)
- **Orange**: Getting old (15-20 minutes)
- **Red**: Urgent! (20+ minutes)

**Item Status Colors**:
- **Gray**: Not started
- **Blue**: Being prepared
- **Green**: Ready/Completed

#### Order Organization
Orders are typically displayed:
- Newest at the top or left
- Sorted by time received
- Grouped by status
- Priority orders highlighted

### Real-Time Updates
The KDS updates automatically:
- New orders appear instantly
- No need to refresh
- Audio alerts for new orders (if enabled)
- Visual notifications
- Status changes sync immediately

### Audio Notifications
When enabled:
- Beep or chime for new orders
- Different sounds for priorities
- Can be adjusted or muted
- Ask manager for sound settings

## Managing Order Items

### Viewing Order Details

Click on any order card to see:
- **Complete Item List**: All kitchen items
- **Quantities**: How many of each item
- **Special Instructions**:
  - Cooking preferences (well-done, rare, etc.)
  - Modifications (no onions, extra sauce)
  - Allergies (NUT ALLERGY, dairy-free)
  - Guest preferences
- **Prep Area**: Confirms it's for kitchen (not bar)
- **Waiter**: Who took the order
- **Order Time**: When submitted

### Understanding Item Information

Each item shows:
- **Item Name**: Menu item name
- **Quantity**: Number needed (e.g., "x2")
- **Special Instructions**: Below item name
- **Status**: Current state (Pending, Preparing, Ready)
- **Action Buttons**: To update status

### Special Instructions

Pay close attention to:
- **Allergies**: Critical for safety
  - Example: "ALLERGY: Shellfish"
  - Take extra precautions
  - Prevent cross-contamination
- **Modifications**:
  - "No onions"
  - "Extra spicy"
  - "Sauce on the side"
- **Cooking Level**:
  - Rare, medium-rare, medium, medium-well, well-done
  - Specific temperatures
- **Substitutions**:
  - "Fries instead of rice"
  - "Gluten-free bread"

**Critical Rule**: Always read special instructions before starting preparation!

## Order Status Updates

### Item Status Workflow

Each item goes through these statuses:

#### 1. Pending (Gray)
- Item just received
- Not yet being worked on
- Awaiting your attention
- Click to start preparing

#### 2. Preparing (Blue)
- You're actively cooking
- Item in progress
- Visible to waiters and managers
- Continue until complete

#### 3. Ready (Green)
- Item is cooked and plated
- Ready for waiter pickup
- Waiter notified
- Waiting for delivery to table

### Updating Item Status

#### Starting an Item
1. Click on the item card
2. Click "Start Preparing" button
3. Status changes to "Preparing" (Blue)
4. Timer starts tracking
5. Begin cooking the item

**When to Start**:
- When you begin actual preparation
- Not before ingredients are ready
- Helps track kitchen timing
- Informs front-of-house

#### Marking Item Ready
1. Item is fully cooked and plated
2. Quality check complete
3. Click "Mark as Ready" button
4. Status changes to "Ready" (Green)
5. Waiter receives notification
6. Place in pickup area

**When to Mark Ready**:
- Food is fully cooked
- Properly plated
- Passed quality check
- Ready for immediate delivery

### Order Completion

#### When All Items Ready
- All items in order marked "Ready"
- Order automatically marked as "Ready"
- Waiter sees entire order ready
- All items should go out together

**Coordinating Multiple Items**:
- Time items to finish together
- Keep hot food hot
- Cold food cold
- Coordinate with other stations

### Removing Orders from Display

Orders automatically disappear when:
- Waiter marks as "Delivered"
- All items are served
- Order is complete

**If order still shows**:
- Check all items marked ready
- Verify waiter picked up
- Contact front-of-house if stuck

## Best Practices

### Efficiency Tips

#### Order Management
1. **Read Completely First**:
   - Review entire order before starting
   - Note special instructions
   - Plan preparation sequence
   - Gather all ingredients

2. **Update Status Promptly**:
   - Mark "Preparing" when you start
   - Mark "Ready" immediately when done
   - Accurate status helps everyone
   - Keeps workflow smooth

3. **Batch Similar Items**:
   - Cook multiple of same item together
   - Efficient use of equipment
   - Consistent quality
   - Faster service

4. **Keep Station Organized**:
   - Clean as you go
   - Prep ingredients ahead
   - Tools within reach
   - Efficient layout

### Quality Control

#### Before Marking Ready
Check:
- ✓ Cooked to correct temperature
- ✓ Proper portion size
- ✓ Special instructions followed
- ✓ Plating is attractive
- ✓ Correct garnishes
- ✓ Clean plate edges
- ✓ Food is hot/cold as required

#### Consistency
- Follow recipes exactly
- Measure portions
- Standard cooking times
- Uniform plating
- Restaurant standards maintained

### Food Safety

#### Critical Points
1. **Allergies**:
   - Take seriously
   - Clean surfaces and tools
   - Prevent cross-contamination
   - Use separate equipment if needed

2. **Temperature**:
   - Cook to safe temperatures
   - Keep hot food hot (above 140°F/60°C)
   - Keep cold food cold (below 40°F/4°C)
   - Use food thermometer

3. **Cross-Contamination**:
   - Separate raw and cooked
   - Clean cutting boards
   - Change gloves
   - Wash hands frequently

4. **Hygiene**:
   - Proper handwashing
   - Clean uniform
   - Hair restraint
   - No jewelry
   - Follow health codes

### Time Management

#### During Slow Periods
- Prep ingredients
- Clean and organize
- Review menu items
- Practice efficiency
- Help other stations

#### During Rush
- Focus on active orders
- Minimize distractions
- Steady, consistent pace
- Ask for help if overwhelmed
- Stay calm

## Troubleshooting

### Common Issues

#### Order Not Appearing on Display
**Problem**: Waiter says order submitted but not showing
**Solutions**:
1. Refresh the display (F5)
2. Check if filtering is hiding orders
3. Verify you're looking at correct date
4. Check internet connection
5. Contact manager

#### Cannot Update Item Status
**Problem**: Buttons don't work or status won't change
**Solutions**:
1. Click directly on button
2. Wait a moment for processing
3. Check internet connection
4. Refresh page
5. Try logging out and back in
6. Contact manager

#### Display Not Auto-Updating
**Problem**: Need to refresh manually for new orders
**Solutions**:
1. Check internet connection
2. Verify WebSocket connection (ask manager)
3. Clear browser cache
4. Try different browser
5. Restart display device
6. Contact IT/manager

#### Wrong Items Showing
**Problem**: Bar items appearing on kitchen display
**Solutions**:
1. Verify item prep area setting
2. Inform manager
3. Ignore bar items
4. Manager will fix item configuration

#### Order Stuck on Display
**Problem**: Completed order not disappearing
**Solutions**:
1. Verify all items marked "Ready"
2. Check if waiter marked as "Delivered"
3. Contact waiter
4. Ask manager to check order status

### Display Issues

#### Screen Frozen
1. Check if mouse/touch works
2. Check internet connection
3. Restart browser
4. Restart device
5. Contact manager

#### Can't Read Display
1. Adjust brightness
2. Clean screen
3. Change viewing angle
4. Check display settings
5. Report hardware issue to manager

### Getting Help

#### In-App Help
- Click **?** icon if available
- Hover for tooltips
- Check help page (`/help`)

#### During Service
- Call manager or supervisor
- Use intercom if available
- Send someone to find help
- Don't let orders wait

#### Training
- Request additional training
- Shadow experienced chefs
- Review this guide
- Ask questions freely
