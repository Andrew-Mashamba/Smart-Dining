## Introduction

Welcome to the SeaCliff POS Waiter Guide. As a waiter, you are the front line of customer service, handling orders, payments, and guest interactions through our POS system.

### Your Role
As a **Waiter**, you can:
- Create and manage orders
- Process payments (cash, card, Stripe)
- View and update table status
- Look up guest information
- Access your order history
- Process tips
- Generate receipts

**Note**: You cannot access system settings, staff management, inventory management, or detailed reports. Contact a manager or administrator for these tasks.

## Creating Orders

### Starting a New Order

#### Step 1: Click "Create Order"
1. Navigate to dashboard
2. Click "Create Order" button
3. Order form appears

#### Step 2: Select Table
1. Choose from dropdown of available tables
2. Tables show name and capacity
3. Select the table where guests are seated

**Tip**: If table isn't available in dropdown, it may be marked as unavailable. Ask manager to check table status.

#### Step 3: Guest Information (Optional)
1. Enter guest phone number if known
2. System will look up existing guest
3. Guest name appears if found
4. Leave blank for walk-in customers
5. Loyalty points may be applied automatically

**Guest Lookup Benefits**:
- Track returning customers
- Apply loyalty rewards
- Reference preferences
- Build customer relationships

#### Step 4: Add Menu Items

**Adding Items**:
1. Browse menu by category
2. Click on items to add
3. Set quantity for each item
4. Add special instructions if needed
5. Items appear in order summary

**Categories Available**:
- Appetizers
- Main Courses
- Desserts
- Beverages
- Bar Drinks

**Special Instructions**:
- Click "Add Note" for any item
- Examples: "No onions", "Extra spicy", "Allergy: nuts"
- Instructions appear in kitchen/bar display
- Be clear and concise

#### Step 5: Review Order
Before submitting:
- Check all items are correct
- Verify quantities
- Review special instructions
- Check table assignment
- Confirm guest information

#### Step 6: Submit Order
1. Click "Submit Order"
2. Order is created with "Pending" status
3. Kitchen/Bar receives notification
4. Order appears in your orders list
5. Order number is generated

**After Submission**:
- Order sent to kitchen/bar display
- You can track status in real-time
- Guests will be notified of estimated time
- You can view order details anytime

## Managing Orders

### Viewing Your Orders

#### All Orders List
Navigate to: **Orders** page

**What You See**:
- All your orders (current and historical)
- Order number
- Table assignment
- Status (Pending, Preparing, Ready, Delivered, Paid)
- Order total
- Time created

**Filters**:
- Filter by status
- Search by order number
- Filter by date
- Filter by table

### Order Status Explained

#### Pending
- Order just created
- Sent to kitchen/bar
- Not yet being prepared
- Awaiting kitchen confirmation

#### Preparing
- Kitchen/bar is actively working
- Items being cooked/mixed
- Check estimated time
- Inform guests of progress

#### Ready
- All items completed
- Ready for pickup
- Bring to table immediately
- Update to "Delivered" after serving

#### Delivered
- Food/drinks served to guests
- Guests are dining
- Monitor for additional requests
- Ready for payment when guests finish

#### Paid
- Payment completed
- Order is closed
- Receipt generated
- Table can be cleared

### Order Details

Click any order to view:
- **Order Information**:
  - Order number
  - Date and time
  - Current status
- **Table & Guest**:
  - Table name
  - Guest name/phone (if recorded)
- **Items**:
  - All items ordered
  - Quantities
  - Prices
  - Special instructions
  - Individual item status
- **Totals**:
  - Subtotal
  - Tax
  - Total amount
  - Payment status
  - Tips

### Updating Order Status

#### Mark as Delivered
After serving food to table:
1. Navigate to order details
2. Click "Mark as Delivered"
3. Status updates to "Delivered"
4. Order ready for payment

**Important**: Only mark delivered after actually serving guests!

#### Handling Issues
If there are problems:
- Wrong items prepared
- Missing items
- Guest complaints

**What to Do**:
1. Don't mark order as delivered
2. Speak with kitchen/bar staff
3. Contact manager if needed
4. Keep guests informed
5. Update once resolved

## Processing Payments

### When to Process Payment
- Guests request the check
- All items are delivered
- Order status is "Delivered"
- Guests are ready to leave

### Payment Options

#### Cash Payment
1. Navigate to order details
2. Click "Process Payment"
3. Select "Cash"
4. Enter amount received from guest
5. System calculates change
6. Display change amount
7. Click "Confirm Payment"
8. Give change to guest
9. Order marked as "Paid"

**Cash Handling**:
- Count cash carefully
- Verify amount before confirming
- Give correct change
- Keep cash secure
- Follow restaurant cash policies

#### Card Payment (Stripe)
1. Navigate to order details
2. Click "Process Payment"
3. Select "Card"
4. Enter card details:
   - Card number
   - Expiry date (MM/YY)
   - CVC
5. Click "Process Payment"
6. Wait for authorization
7. Payment confirmed or declined
8. Give receipt to guest

**Card Payment Tips**:
- Ensure stable internet connection
- Double-check card details
- Handle card securely
- Never write down card info
- Give customer receipt

#### Card Terminal (Physical Card Reader)
1. Navigate to order details
2. Click "Process Payment"
3. Select "Card Terminal"
4. Use physical card reader
5. Follow terminal prompts
6. Guest inserts/taps card
7. Wait for approval
8. Mark payment as complete
9. Give receipt

### Processing Tips

#### Adding Tips
During payment:
1. Ask guest if they'd like to add tip
2. Guest specifies amount or percentage
3. Enter tip amount in system
4. Add to total
5. Process payment with tip included
6. Tip recorded to your account

**Tip Options**:
- Percentage (15%, 18%, 20%, custom)
- Fixed amount
- No tip

#### Tip Best Practices
- Always ask politely
- Never pressure guests
- Accept tip preference gracefully
- Thank guests for tips
- Ensure tips recorded accurately

### Generating Receipts

#### Automatic Receipt
- Generated automatically after payment
- Includes all items, prices, tax, tips
- Can be printed or emailed

#### Manual Receipt Generation
1. Go to order details
2. Click "Generate Receipt"
3. PDF downloads
4. Print or email to guest

**Receipt Contains**:
- Restaurant information
- Date and time
- Order number
- All items with prices
- Subtotal
- Tax breakdown
- Tips
- Total amount
- Payment method
- Your name (waiter)

### Split Payments

If guests want to split payment:
1. Calculate individual amounts
2. Process first payment
3. Create note about split
4. Contact manager for assistance
   - Current system processes one payment per order
   - Manager can help handle splits

**Workaround**:
- Calculate splits manually
- Process payments separately
- Keep careful records
- Document in order notes

## Troubleshooting

### Common Issues

#### Cannot Create Order
**Problem**: Order form won't submit
**Solutions**:
1. Check all required fields filled
2. Verify table is selected
3. Ensure at least one item added
4. Check internet connection
5. Try refreshing page
6. Contact manager if persists

#### Payment Processing Fails
**Problem**: Card payment declined or errors
**Solutions**:
1. Verify card details entered correctly
2. Check internet connection
3. Try processing again
4. Use alternative payment method
5. Contact manager for help
6. Check Stripe status with manager

#### Order Not Appearing in Kitchen
**Problem**: Kitchen didn't receive order
**Solutions**:
1. Check order actually submitted
2. Verify order status is not "Draft"
3. Check kitchen display is on
4. Notify kitchen staff manually
5. Contact manager

#### Cannot Find Guest
**Problem**: Guest lookup returns no results
**Solutions**:
1. Verify phone number is correct
2. Try searching by name
3. Guest may not be in system yet
4. Create order without guest
5. Ask manager to create guest profile

#### Wrong Order Total
**Problem**: Total doesn't match expected
**Solutions**:
1. Review all items added
2. Check quantities
3. Verify prices with menu
4. Consider tax calculation
5. Contact manager to verify

#### Table Shows as Unavailable
**Problem**: Can't select table for order
**Solutions**:
1. Check if table has active order
2. Verify table status
3. Ask manager to check table
4. Use alternative table
5. Wait for table to become available
